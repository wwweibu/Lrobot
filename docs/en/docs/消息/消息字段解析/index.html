消息字段解析

BILI 和 WECHAT 的发送机制为: 若发送的内容匹配到了某个表情，则在发送后被替换成该表情 如文字发送 [doge_金箍] 或者 /::) 会被 B 站和微笑自动解析成金箍表情和微笑
